Say Hello to Migo!

The way people used computers yesterday, today.

(But easier)

What is Migo?

Migo is an interactive easy-to-use command prompt that anyone can use. No programming knowledge necessary.

Why do I need Migo? Can't I just click on stuff?

Maybe, but using Migo allows you to develop a better understanding of how computers work and allows you to increase your functionality when using a computer.

So how does Migo work?

Migo does whatever you tell it to, for example, if you were to tell Migo "Listen to music", Migo would open iTunes.

What OS does Migo work on?

As of right now Migo is in testing phases for OSX, once Migo for Mac is stable, I will begin to work on versions for Linux and Windows.
